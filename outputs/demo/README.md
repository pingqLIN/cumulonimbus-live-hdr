# outputs/demo

本目錄保留專案對外或展示用的 DEMO 輸出。除非特別標註，日常 render/render-loop 測試輸出不納入 Git。

版本命名建議：

- `outputs/demo/<period>/cumulonimbus-quick-hdr.mp4`
- `outputs/demo/<period>/cumulonimbus-test-hdr.mp4`
- `outputs/demo/<period>/cumulonimbus-demo-loop.mp4`
- 搭配 `outputs/demo/<period>/metrics/*.json` 放同時期的參數/指標。

`<period>` 可用 `2026-06`、`v1`、`live-episode-01` 等可追溯名稱。
